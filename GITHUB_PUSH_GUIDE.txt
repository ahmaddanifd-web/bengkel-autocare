üîê AUTHENTICATION
=================

Jika push dengan HTTPS password deprecated:
1. Generate Personal Access Token:
   https://github.com/settings/tokens
   
2. Select scopes: repo, read:user
   
3. Copy token dan gunakan sebagai password saat push
   
ATAU lebih mudah dengan SSH:
1. Generate SSH key:
   ssh-keygen -t ed25519 -C "your_email@example.com"
   
2. Add ke GitHub:
   https://github.com/settings/ssh/new
   
3. Use SSH URL:
   git remote add origin git@github.com:YOUR_USERNAME/bengkel-autocare.git
