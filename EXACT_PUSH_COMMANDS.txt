JIKA PUSH GAGAL - TROUBLESHOOTING
==================================

Error 1: "fatal: A branch named 'master' already exists"
Fix: git branch -D master
     git branch -M main
     git push -u origin main

Error 2: "fatal: 'origin' does not appear to be a 'git' repository"
Fix: git remote add origin https://github.com/YOUR_USERNAME/bengkel-autocare.git
     git push -u origin main

Error 3: "fatal: unable to access 'https://github.com/...': The requested URL returned error: 401"
Fix: Pastikan Personal Access Token benar:
     https://github.com/settings/tokens
     Cek scopes: repo, read:user

Error 4: "rejected (fetch first)"
Fix: git pull origin main --allow-unrelated-histories
     git push -u origin main

Error 5: File too large
Fix: Update .gitignore dan gunakan Git LFS untuk file besar:
     https://git-lfs.github.com/

   âœ“ repo (full control)

SETUP SSH (Optional - Lebih secure)
====================================

1. Generate SSH key:
   ssh-keygen -t ed25519 -C "your.email@gmail.com"
   
   Tekan Enter 3x (default location, no passphrase)

2. Display public key:
   type $env:USERPROFILE\.ssh\id_ed25519.pub

3. Add ke GitHub:
   - Go to: https://github.com/settings/ssh/new
   - Title: "My Windows PC"
   - Key: Paste isi public key
   - Click "Add SSH key"

4. Test connection:
   ssh -T git@github.com
   
   Output harusnya:
   "Hi YOUR_USERNAME! You've successfully authenticated..."

5. Update remote (jika sudah ada):
   git remote set-url origin git@github.com:YOUR_USERNAME/bengkel-autocare.git

6. Push tanpa password:
   git push -u origin main
