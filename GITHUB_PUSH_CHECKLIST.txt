üì± ALTERNATIVE: Setup SSH (Lebih Secure & Convenient)
===================================================

Tidak perlu input password setiap kali push.

1. Generate SSH Key:
   ssh-keygen -t ed25519 -C "your.email@gmail.com"
   
   Tekan Enter 3x untuk default location & no passphrase

2. Add Public Key ke GitHub:
   
   a. Copy content dari file:
      type $env:USERPROFILE\.ssh\id_ed25519.pub
   
   b. Go to: https://github.com/settings/ssh/new
   
   c. Title: "My Windows PC"
      Key: Paste isi file public key
   
   d. Click "Add SSH key"

3. Update remote URL:
   git remote set-url origin git@github.com:YOUR_USERNAME/bengkel-autocare.git

4. Test connection:
   ssh -T git@github.com
   
   Output: "Hi YOUR_USERNAME! You've successfully authenticated..."

Setelah itu, push tanpa perlu authentication:
   git push
